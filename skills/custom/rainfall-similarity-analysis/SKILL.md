# 降雨图斑相似性分析技能

本技能执行降雨图斑相似性分析全流程，将"当前降雨"与历史相似降雨的"图斑图片"进行比对，找出最相似的历史降雨图斑，并支持查看选中相似雨时段的水文信息。

## 适用场景

- 用户提出"图斑相似性分析"、"图斑比对"、"相似雨分析"、"降雨图斑相似度分析"等需求时激活
- 用于洪水预报、防汛研判中寻找与当前降雨形势最接近的历史降雨案例

## 核心工具

| 工具名 | 用途 |
|-------|------|
| `rainfall_similarity_analysis` | 图斑相似性分析统一入口，按 `step` 参数执行不同阶段 |

> 说明：本技能通过一个工具的 `step` 参数控制流程阶段（`step=1` 列表查询与选择确认；`step=2` 链式执行详情→相似雨→图斑→图斑相似度；`step=3` 查询选中相似雨时段的水文信息并整合图斑图片）。

## 整体流程

```
[step=1] 查询各流域最新降雨情况
            ↓
       返回流域+降雨列表，询问用户选择
            ↓
[step=2] 接收用户选择的 流域 + 降雨编号
            ↓
       2.1 查询该场降雨详情
            ↓
       2.2 将降雨详情作为参数计算相似雨
            ↓
       2.3 拿到相似雨结果中的降雨时间，查询每场相似雨的降雨图斑图片
            ↓
       2.4 将图斑图片作为参数调用图斑相似度计算模型，获取最相似的3场图斑
            ↓
       返回最相似的3场降雨时间段，询问用户选择查看哪一场
            ↓
[step=3] 接收用户选择的 流域 + 起止时间 + 图斑URL
            ↓
       3.1 调用水文接口查询该时段水文数据
            ↓
       返回水文信息 + 图斑图片URL整合结果
```

## 第一步：查询各流域最新降雨情况（step=1）

### 触发方式

用户首次提出图斑相似性分析需求时，模型调用 `rainfall_similarity_analysis(step="1")`。

### 工具调用

```python
rainfall_similarity_analysis(step="1")
```

无入参。

### 期望返回（接口入参/出参待用户确认）

```json
{
  "success": true,
  "basins": [
    {
      "basin_name": "伊洛河",
      "basin_code": "...",
      "rainfall_events": [
        {
          "rainfall_id": "RF-2026-0601-001",
          "start_time": "2026-05-30 08:00:00",
          "end_time": "2026-06-01 12:00:00",
          "total_rainfall": 86.5,
          "description": "..."
        }
      ]
    }
  ],
  "message": "已查询到各流域最新降雨情况，请选择目标降雨事件"
}
```

### 模型行为

1. 调用 `rainfall_similarity_analysis(step="1")` 获取各流域最新降雨情况
2. **明确告知用户**：当前返回的是各流域**最新一场降雨**信息
3. **向用户发起选择问询**（使用 AskUserQuestion 工具）：
   - 列出可选流域及对应的降雨事件
   - 让用户选择一个流域 + 一场降雨（用降雨编号 `rainfall_id` 标识）
4. 等待用户确认后，进入第二步

---

## 第二步：链式执行详情→相似雨→图斑→图斑相似度（step=2）

### 触发方式

用户在第一步选择完流域和降雨事件后，模型调用 `rainfall_similarity_analysis(step="2", basin=..., rainfall_id=...)`。

### 工具调用

```python
rainfall_similarity_analysis(
    step="2",
    basin="<用户选择的流域名称>",
    rainfall_id="<用户选择的降雨编号>"
)
```

### 内部执行步骤（按顺序执行，不可调整顺序）

| 序号 | 步骤 | 描述 | 入参 | 出参 |
|------|------|------|------|------|
| 2.1 | 查询降雨详情 | 根据降雨编号查询该场雨的详细过程 | `rainfall_id` | 降雨过程详情（时段、累计雨量、时空分布等） |
| 2.2 | 计算相似雨 | 将降雨详情作为参数计算历史相似雨 | 降雨详情 | 多场历史相似雨结果（含每场雨的降雨时间） |
| 2.3 | 查询图斑图片 | 用每场相似雨的降雨时间查询对应图斑图片 | 相似雨结果中的降雨时间 | 每场相似雨的图斑图片 |
| 2.4 | 图斑相似度计算 | 将图斑图片作为参数调用图斑相似度计算模型 | 图斑图片 | 最相似的图斑结果 |

### 期望返回

```json
{
  "success": true,
  "basin": "伊洛河",
  "rainfall_id": "RF-2026-0601-001",
  "steps": {
    "rainfall_detail": { /* 第 2.1 步结果 */ },
    "similar_rainfall": [ /* 第 2.2 步结果（多场历史相似雨） */ ],
    "rainfall_raster_images": [ /* 第 2.3 步结果（每场相似雨对应一张图斑） */ ],
    "most_similar_raster": { /* 第 2.4 步结果（最相似图斑） */ }
  },
  "message": "图斑相似性分析完成，最相似的历史降雨图斑为 ..."
}
```

### 模型行为

1. 调用 `rainfall_similarity_analysis(step="2", basin=..., rainfall_id=...)`
2. 工具内部按顺序执行 2.1 → 2.2 → 2.3 → 2.4
3. 拿到完整结果后，向用户汇报：
   - 选中的降雨事件信息（流域、降雨编号、时间、累计雨量）
   - 匹配到的历史相似雨数量
   - 最相似的3场降雨时间段列表
4. **明确告知用户**返回了3场最相似的降雨时间段
5. **向用户发起选择问询**（使用 AskUserQuestion 工具）：
   - 列出3场降雨的起止时间
   - 让用户选择要查看详细水文信息的那一场
6. 等待用户确认后，进入第三步

## 第三步：查询水文信息并整合图斑图片（step=3）

### 触发方式

用户在第二步选择完要查看的相似雨时间段后，模型调用 `rainfall_similarity_analysis(step="3", basin=..., start_time=..., end_time=..., raster_image_url=...)`。

### 工具调用

```python
rainfall_similarity_analysis(
    step="3",
    basin="<流域名称>",
    start_time="<用户选择的相似雨起始时间>",
    end_time="<用户选择的相似雨结束时间>",
    raster_image_url="<对应时段的图斑图片URL>"
)
```

### 入参说明

| 参数 | 说明 | 来源 |
|------|------|------|
| `basin` | 流域名称 | 从 step=2 结果中获取 |
| `start_time` | 用户选择的那场相似雨起始时间（含时分秒，如 `2013-09-21 00:00:00`） | 用户选择 |
| `end_time` | 用户选择的那场相似雨结束时间（含时分秒，如 `2013-09-22 00:00:00`） | 用户选择 |
| `raster_image_url` | 对应时段的图斑图片URL | 从 step=2 结果的 `rainfall_raster_images` 中提取 |

### 内部执行步骤

| 序号 | 步骤 | 描述 |
|------|------|------|
| 3.1 | 查询水文信息 | 调用水文接口 `getByBasinNameListAndTime`，传入流域名称和起止时间 |

### 模型行为

1. 调用 `rainfall_similarity_analysis(step="3", basin=..., start_time=..., end_time=..., raster_image_url=...)`
2. 工具内部调用水文接口并整合图斑图片URL
3. 向用户展示结果，包含：
   - 选中相似雨时段的水文数据（各测站流量过程）
   - 对应时段的图斑图片URL
   - 将图斑图片URL嵌入为可点击查看的链接

## 工具参数详细说明

### `rainfall_similarity_analysis`

| 参数 | 说明 | 是否必填 | 适用阶段 |
|------|------|---------|---------|
| `step` | 流程阶段，可选值：`"1"`（查询最新降雨）/ `"2"`（链式执行详情→相似雨→图斑→图斑相似度）/ `"3"`（查询水文信息并整合图斑图片） | 是 | 全部 |
| `basin` | 流域名称，例如："黄河"、"伊洛河"、"洛河" | 是（step=2/3 时） | step=2, step=3 |
| `rainfall_id` | 降雨编号，由 step=1 接口返回 | 是（step=2 时） | step=2 |
| `start_time` | 查询起始时间（格式：`2013-09-21 00:00:00`） | 是（step=3 时） | step=3 |
| `end_time` | 查询结束时间（格式：`2013-09-22 00:00:00`） | 是（step=3 时） | step=3 |
| `raster_image_url` | 对应时段的图斑图片完整URL | 是（step=3 时） | step=3 |

---

## 注意事项

1. **三步流程不可跳步**：必须先 step=1 拿到降雨列表，让用户选择后再 step=2，拿到相似雨时间段后再 step=3
2. **用户选择不可省略**：第一步必须明确询问用户选择哪个流域的哪场雨；第二步后必须询问用户选择哪场相似雨查看详情
3. **第二步内部步骤固定**：2.1→2.2→2.3→2.4 顺序执行，不可调整顺序
4. **step=3 的图斑URL**：必须从 step=2 返回的 `rainfall_raster_images` 数据中提取与用户选中时间段匹配的 `contourImage` 地址
5. **接口细节后续完善**：当前每一步的接口入参/出参细节尚未确定，本文档标记为"待用户确认"，后续用户提供接口文档后需补充
