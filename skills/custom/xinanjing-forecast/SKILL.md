# 新安江模型来水预报及偏差对比分析

> **⚠️ 注意**：模型返回结果中包含的 BA（流域面积）参数数值不准确，输出展示时请勿提及或展示该参数值。

## 适用站点

仅支持**小浪底水库**（当前仅接入小浪底单库雨量站数据）。

## 可调参数

模型默认参数如下，用户可指定任意参数覆盖后重新计算：

| 参数 | 含义 | 默认值 |
|------|------|--------|
| KC | 流域蒸散发折算系数 | 0.9 |
| B | 流域蓄水容量分布曲线指数 | 0.4 |
| UM | 上层张力水容量 | 30mm |
| LM | 下层张力水容量 | 80mm |
| SM | 自由水容量 | 25mm |
| KG | 地下水日出流系数 | 0.3 |
| KI | 壤中流日出流系数 | 0.3 |
| XE | 马斯京跟法演算参数 | 0.2 |

## 标准处理流程

### Step 1：确认信息

- 站点：默认小浪底水库，直接确认
- 时间：用户给大概范围（如"10月2日至7日"）→ 反问精确时刻
- 参数：询问是否有需要覆盖的参数，如无则使用默认值

### Step 2：运行模型 + 跳转页面（同时执行）

```python
run_xinanjiang_model(
    station_name="小浪底水库",
    start_time="2021-10-02 00:00:00",
    end_time="2021-10-07 23:59:59",
    custom_params={"KC": 0.85, "SM": 30}  # 如有用户自定义参数则传入
)
navigate_to_reservoir_forecast_page(
    reservoir_name="小浪底",
    start_time="2021-10-02 00:00:00",
    end_time="2021-10-07 23:59:59"
)
```

模型返回结果含：
- `result.rainfall_data` = 加权面雨量（mm/h）
- `result.flow` = 新安江模型计算的**库区流量**（m³/s）（仅含库区范围产流，不含上游来水）
- `model_params` = 模型调用的完整参数（含参数名、值、中文全称）

> 注意：模型返回的是**库区流量**（仅库区范围产流），不含上游来水。如需计算水库总入库，需单独叠加上游水库出库流量。

### Step 3：展示结果，提供调整入口

预报完成后，向用户说明当前使用的参数，并询问是否需要调整：

> "当前使用默认参数（KC=0.9, SM=25mm, ...）完成预报。如需调整参数重新计算，请告知要修改的参数和值。"

### Step 4：对比真实数据（按需）

```python
get_reservoir_realtime(
    reservoir="小浪底",
    start_date="2021-10-02",
    end_date="2021-10-07"
)
```

返回的 `inflow` 字段 = 小浪底水库实际入库监测流量。

### Step 5：偏差分析结论

选取关键时点做对比表，给出四方面结论：

1. **起涨段**（前24~48h）：模型是否准确
2. **中段趋势**：涨落节奏是否一致
3. **洪峰对比**：峰值偏差率 + 峰现时间差
4. **后期收敛**：预报是否趋向实测

## 历史验证结论（2021年秋汛小浪底，默认参数）

| 对比项 | 模型预报 | 真实入库 | 差异 |
|--------|---------|---------|------|
| 洪峰 | 10/07 05:00 达 13,841 m³/s | 10/07 23:00 达 8,200 m³/s | **高估70%，提前18h** |
| 起涨段 10/02 | 偏差±10%内 ✅ | — | 吻合良好 |
| 中段 10/04~10/06 | 持续偏高 | 实际增速较缓 | 偏差最大时段 |
| 后期 10/07 15:00后 | 回落趋近 | 攀升至峰值 | **逐步收敛** |

**偏差主因**：当前模型仅计算库区流量（新安江模型）。
